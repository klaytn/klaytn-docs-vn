Hướng dẫn này cung cấp hướng dẫn từng bước để thiết lập mạng lưới Chuỗi dịch vụ và kết nối mạng lưới đó với mạng lưới Baobab Klaytn. Bạn cũng sẽ biết cách kích hoạt neo định kỳ và chuyển giá trị chuỗi chéo. Để vận hành các dịch vụ blockchain thực tế với Chuỗi dịch vụ, chúng ta cần giải thích về Tính sẵn sàng cao, xây dựng Chuỗi dịch vụ theo bậc và chuyển giao giá trị giữa các Chuỗi dịch vụ kết nối.
